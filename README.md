# 运行方法：
默认是多分类（包括二分类）任务，修改config里面的相关配置即可直接运行

config中包括数值特征、类别特征、不定长特征（如：{"兴趣":"篮球、音乐、旅游"}。一个记录项对应几个item的特征，注意分隔符为','）的处理方案。只需要把对应的columns写上去就可以了
